Introduction to GRASS GIS

Software and data

  • GRASS GIS 7.0
  • GRASS GIS NC SPM sample dataset: zip or tar.gz (installer on MS Windows offers to download it doing installation)
  • Uncompress the downloaded file and put the content grassdata directory in your home directory or directory of your choice. Note to MS Windows users: Check if the extraction created just one nc_spm_08_grass7 directory. If it has subdirectory with the same name, copy just the subdirectory.
